## Usage in Documentation

To reference images in Markdown:

```markdown
![Description](../images/filename.png)
```

Example:
```markdown
![Nexus Rover Front View](../images/rover_front.jpg)
```

## Image Guidelines

- **Format:** JPG for photos, PNG for diagrams/screenshots, GIF for animations
- **Resolution:** At least 1200px wide for main images
- **Size:** Optimize to keep under 500KB each
- **Naming:** Use lowercase with underscores (e.g., `wiring_diagram.png`)
- **Quality:** Clear, well-lit photos; high-contrast diagrams

## Contributing Images

When contributing images:
1. Ensure good lighting and focus
2. Include scale reference if helpful (ruler, coin)
3. Annotate diagrams with labels
4. Compress large files before committing
5. Update this README with description
